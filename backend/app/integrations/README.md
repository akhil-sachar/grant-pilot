# Integrations

These adapters define where external systems connect.

- ClickHouse: analytical persistence and event history
- Composio: tool execution (Gmail, Google Docs, Calendar, Drive)
- OpenAI: agent generation and scoring
- Langfuse: LLM observability and tracing
- Guild AI: experiment tracking for agent runs
- OpenUI: dynamic UI layout generation

Funding opportunity ingestion uses built-in source adapters (Grants.gov, NSF, NIH, etc.) — no external sync pipeline required.
